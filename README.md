## :wrench: Installation

1. Clone this repository

```bash
$ git clone https://github.com/TeamNeo2021/gigandjob-backend.git
```

2. Install al the dependencies

```bash
$ flutter pub get
```

## :rocket: Running the app

```bash
# development
$ flutter run lib/main.dart

# test
$ flutter test

```
